# Communication spectrum

  • Radio waves:
    • 30 MHz ~ 1 GHz
    • 可以從電離層反射，傳播距離很遠
    • 適合 Omnidirectional transmission: 信號在所有方向上均勻傳播
  • Microwaves:
    • 1 GHz ~ 40 GHz
    • 通常用於點對點通信，例如衛星通信和微波鏈路。
    • 容易受到障礙物的影響，傳播距離較短。
  • Infrared:
    • 從放熱物體發出的電磁波。
    • Line of sight transmission: 信號需要直接視線才能傳播，容易被障礙物阻擋。
    • 適合在特定區域內點對點通信，例如遙控器和近距離無線通信。

# Spreading spectrum

  • 由 International Telecommunication Union (ITU) 定義
  • 要得到一個頻率範圍的使用權，通常有以下幾種方式：
    • Lottery
    • Auction
    • Comparative bidding
  • 有些頻率範圍是專用的，有些頻率範圍是免費的
    • licensed spectrum，需要向政府申請並支付費用才能使用。
      • Cellular system
      • Paging system
      • Personal communication system
    • unlicensed spectrum，任何人都可以使用
      • ISM (Industrial, Scientific, and Medical) bands
      • 2.4 GHz band: Wi-Fi, Bluetooth, Zigbee
  • Spread spectrum: 一種將信號分散在較寬頻率範圍內的技術，可以提高抗干擾能力和安全性。
    • Frequency hopping spread spectrum (FHSS):
      • 對噪音和干擾具有更強的抵抗力，因為信號在不同頻率之間跳躍，使得干擾者難以鎖定和干擾信號

# Modulation schemes

  • Amplitude modulation (AM): 通過改變信號的振幅來傳輸信息
  • Frequency modulation (FM): 通過改變信號的頻率來傳輸信息

# Digital modulation schemes

在無線通信中，數字調製方案用於將數字數據轉換為適合無線傳輸的信號形式。常見的數字調製方案包括：

  • Amplitude Shift Keying (ASK): 通過改變信號的振幅來表示不同的數字值，例如 0 和 1。
  • Frequency Shift Keying (FSK): 通過改變信號的頻率來表示不同的數字值，例如 0 和 1。
  • Phase Shift Keying (PSK): 通過改變信號的相位來表示不同的數字值
    • Binary Phase Shift Keying (BPSK): 使用兩個相位來表示 0 和 1，在衛星通信中常用
    • Quadrature Phase Shift Keying (QPSK): 使用四個相位來表示 00、01、10 和 11，比 BPSK 更有效率，常用於 Wi-Fi 和 LTE 等無線通信系統
    • Quadrature Amplitude Modulation (QAM): 結合振幅和相位的變化來表示更多的數字值，例如 16-QAM、64-QAM 等
    • Differential Phase Shift Keying (DPSK): 通過比較當前信號的相位與前一個信號的相位來表示數字值，避免了對相位同步的需求
    • π/4\pi/4π/4-DPSK: 一種特殊的 DPSK，使用四個相位來表示數字值，提供更好的性能和抗干擾能力

# Wireless propagation model

  • Free-space path loss: 1/r2 1/r41/r^2~1/r^41/r2 1/r4 的衰減，其中rrr 是傳輸和接收的距離。
  • Doppler shift: 當發射器和接收器之間存在相對運動時，接收的信號頻率會發生變化。
  • Slow fading: 由於環境變化較慢，導致信號強度緩慢變化。
    • Reflection: 當信號遇到大於波長的物體時產生反射現象，導致信號強度減弱。
    • Scattering: 當信號遇到小於波長的物體時產生散射現象，導致信號強度減弱。
    • Diffraction / Shadowing: 當信號被障礙物阻擋時，會產生衍射現象導致信號強度減弱。
  • Fast fading: 由於多徑效應，信號在不同路徑上到達接收器，導致信號有可能被增強或抵消，信號強度快速變化。
